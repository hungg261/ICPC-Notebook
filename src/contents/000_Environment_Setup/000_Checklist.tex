% Checklist \& Code::Blocks Setup

Contest preparation, environment validation, and shortcuts.

\textbf{Contest Preparation \& Environment Verification:}
\begin{itemize}
    \item Bring ID, credentials, pens, and printed Team Notebook.
    \item Verify system time, keyboard layout, and workspace comfort.
    \item Enable C++17/14: \lstinline|Settings -> Compiler -> Compiler Settings -> Flag "-std=c++17"|.
    \item Add compiler optimization \& warning flags: \lstinline|-O3|, \lstinline|-Wall|, \lstinline|-Wextra|, \lstinline|-Wshadow|.
    \item Enable auto-save: \lstinline|Settings -> Environment -> Autosave -> Check every 1 min|.
    \item Snippets: \lstinline|Settings -> Editor -> Abbreviations|. Expands with \lstinline|Ctrl + J|.
\end{itemize}

\textbf{Essential Code::Blocks Shortcuts:}
\begin{itemize}
    \item \lstinline|Ctrl + Shift + C|: Comment selected block.
    \item \lstinline|Ctrl + Shift + X|: Uncomment selected block.
    \item \lstinline|Ctrl + F9| / \lstinline|Ctrl + F10| / \lstinline|F9|: Compile / Run / Both.
    \item \lstinline|Ctrl + Space| / \lstinline|Ctrl + Shift + Space|: Autocomplete / Parameter tips.
    \item \lstinline|Alt + G|: Jump to declaration or definition.
    \item \lstinline|Alt + Up/Down|: Move selected lines up or down.
    \item \lstinline|Ctrl + E|: Select next occurrence of keyword (Multi-select).
    \item \lstinline|Ctrl + Click|: Multi-edit at custom positions.
    \item \lstinline|Ctrl + D|: Duplicate selected text or current line.
    \item \lstinline|Ctrl + T|: Swap current line with the line above it.
\end{itemize}
