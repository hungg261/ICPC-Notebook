\begin{itemize}
    \item Highly Composite Numbers tables are crucial for finding numbers with the maximum number of divisors under a specific upper bound.
    \item Max values for standard types: $12!$ fits within standard 32-bit signed \lstinline|int| (since $13!$ overflows), while $20!$ is the largest factorial fitting inside a 64-bit signed \lstinline|long long| (since $21!$ overflows).
    \item For geometry tasks, computing a 2D cross product $A \times B = x_A y_B - y_A x_B$ using 64-bit coordinates requires \lstinline|__int128| to completely safe-guard against intermediate overflow.
\end{itemize}
